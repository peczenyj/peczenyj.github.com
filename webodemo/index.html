demo: prebid.js version 6.6.0 with weborama RTD submodule using different types of configuration


site-centric (contextual) only
user-centric (wam) only
site-centric (contextual) + user-centric (wam)
Site-Centric configuration will use the profile of one page in http://www.laprovence.com with profile: 
	  {
	    webo_ctx: ["Rugby_c3569", "Travel_in_France_c3524"]
          }
User-Centric configuration expects the page loads wamfactory wildcard js tag with wam2gam enabled, it will store on localStorage the following profile: 
	    {
	      webo_cs: ["red"]
	    }
